# SSL Certificates\n\nPlace your SSL certificates here:\n- cert.pem (certificate)\n- key.pem (private key)\n\nFor development, generate self-signed certificates:\n```bash\nopenssl req -x509 -nodes -days 365 -newkey rsa:2048 \\n  -keyout nginx/ssl/key.pem \\n  -out nginx/ssl/cert.pem \\n  -subj "/C=US/ST=State/L=City/O=Organization/CN=localhost"\n```\n\nFor production, use Let's Encrypt:\n```bash\nsudo certbot --nginx -d hermes.yourdomain.com\n```
