Getting Started

Base URL

https://api.hermes.parallax-ai.app/v1

All API requests should be made to this base URL using HTTPS.

Making Your First Request

1. Get your API key from the dashboard

Navigate to your HERMES dashboard and generate an API key in the Settings section.

2. Make a test request

curl -X GET "https://api.hermes.parallax-ai.app/v1/status" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

3. Expected Response

{
  "status": "operational",
  "service": "HERMES AI Voice Agent",
  "version": "1.0.0",
  "timestamp": 1699564800.123
}

Authentication

API Key Authentication

HERMES API uses API key authentication. Include your API key in the Authorization header of every request.

Security Best Practices

  • • Never expose API keys in client-side code
  • • Use environment variables to store API keys
  • • Rotate API keys regularly
  • • Use different API keys for different environments
# Include in every request
Authorization: Bearer hermes_live_sk_1234567890abcdef

WS /ws/voice

Real-time Voice Processing

WebSocket connection for real-time voice processing with low latency streaming. 

// Connect to WebSocket
const ws = new WebSocket('wss://api.hermes.parallax-ai.app/ws/voice');

// Send audio data
ws.send(JSON.stringify({
  type: 'audio',
  data: audioBase64,
  session_id: 'session_123'
}));

// Receive real-time transcription
ws.onmessage = function(event) {
  const response = JSON.parse(event.data);
  console.log('Transcription:', response.transcription);
};

Webhooks

Event Notifications

HERMES can send real-time webhook notifications to your application when important events occur.

Supported Events

call.started New voice call initiated
call.completed Voice call processing completed
lead.qualified Lead qualified and requires follow-up
integration.sync Data synchronized with external system

Example Webhook Payload

{
  "id": "evt_1234567890",
  "type": "lead.qualified",
  "created": "2024-01-15T10:30:00Z",
  "data": {
    "lead_id": "lead_abc123",
    "session_id": "session_xyz789",
    "client_info": {
      "name": "John Doe",

    },

    "urgency_score": 8.5,

Python SDK Example

import hermes_api

# Initialize client
client = hermes_api.Client(api_key="your_api_key")

# Process voice input
result = client.voice.process({
    "audio_data": audio_base64,
    "session_id": "session_123",
    "tenant_id": "firm_abc"
})

print(f"Transcription: {result.transcription}")
print(f"Practice Area: {result.legal_analysis.practice_area}")
print(f"Urgency Level: {result.urgency_level}")

# Handle qualified leads
if result.legal_analysis.requires_attorney:
    # Create lead in CRM
    lead = client.leads.create({
        "name": result.client_info.name,
        "phone": result.client_info.phone,
        "practice_area": result.legal_analysis.practice_area,
        "urgency_score": result.urgency_level
    })
    print(f"Lead created: {lead.id}")
